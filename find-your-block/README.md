## 🖥 Acesso ao Projeto

### 🏳 Primeiros Passos
```bash
# Clone o projeto na sua máquina
git clone https://github.com/Gustavo-Nasc/Bora-Codar

# Abra a pasta do "Find Your Block"
cd find-your-block

# Instale as dependências do projeto
npm i
# ou
yarn

```

### Acesso pelo Localhost
```bash
# Inicie o projeto localmente pelo localhost
# A porta do projeto será a 5173 ou a próxima que estiver disponível
npm run dev
# ou
yarn dev
```

## Acesso pelo modo "Preview"
```bash
# Execute o comando que irá Buildar o projeto
npm run build

# Logo após o projeto ser buildado, execute o comando que irá rodar o projeto no modo Preview, ou seja, o projeto buildado
# A porta do projeto será a 4173 ou a próxima que estiver disponível
npm run preview
```
